# Deutsch A1 自學工具

給**中文母語者**的德語 A1 自學工具——單字卡、發音規則、文法、句型例句、測驗、課程導覽，全部裝在一個 HTML 檔裡。**免註冊、零追蹤、離線可用。**

👉 **直接使用：** https://nonee017-design.github.io/deutsch/deutsch-a1.html

## 特色

- **單字卡**：性別用顏色記（der＝藍、die＝紅、das＝綠、複數＝紫），可翻面、發音、逐字母拼讀，標「我會了／再練」
- **文法**：表格化系統卡，先講邏輯再講細節，橘色框標最容易錯的地方
- **測驗**：字義、性別、複數、文法填空、聽力、語序、打字題；答錯的字自動進「不熟」牌堆重點加強；每課有畢業考、每三課一關單元挑戰
- **課程導覽**：每課一篇導讀短文＋重點地圖，點任何德文字可查義
- **發音**：瀏覽器內建德文語音（建議搭配 Forvo 聽真人發音）

## 使用須知

- **進度只存在你這台裝置的瀏覽器**（localStorage）。清除瀏覽資料會清掉進度；工具內建「匯出／匯入」備份功能，建議定期備份。
- 部分內容（導讀短文等）為 **AI 生成、未經母語者校對**，發現錯誤或可疑之處，歡迎開 [Issue](../../issues) 回報。
- 適用範圍：德語 A1（零基礎起步）。內容持續擴充中。
